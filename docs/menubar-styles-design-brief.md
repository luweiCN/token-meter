# TokenMeter 菜单栏额度组件样式族设计需求（给设计师 / OpenDesign）

## 产品背景

TokenMeter 是一个 macOS 菜单栏常驻应用，监控本地 coding agent（Claude Code、Codex CLI 等）的 token 用量、花费与订阅额度。菜单栏里常驻一条组件，从左到右：**每家供应商一个「额度 cell」，最右侧一个「今日用量」数字**。

**本次任务：为这条组件设计多种可切换的样式。** 用户会在设置页自由切换（一种样式看久了会腻），所以请设计 5~8 种**彼此差异明显**的样式。风格方向不设限——条 / 柱 / 点阵 / 徽章 / 仪表 / 纯文字……都可以探索，也完全可以跳出这些词，自由发挥。唯一的要求是：每种样式都必须装得下下面的「元素」、扛得住「定制退化」、活得了「菜单栏环境」。

现有样式是「同心双环」（外环 = 7 天额度、内环 = 5 小时额度，旁挂两个数字如 `Claude ◎ 55·96`），它会作为样式之一保留，不用重新设计，但可以作为密度和气质的参照物。

## 组件要展示的元素（数据模型）

### 每家供应商一个 cell（常驻 1~4 家）

cell 内可用的信息：

- **品牌短名**：Claude / Codex / 智谱（注意有中文名，宽度和英文不同）
- **额度窗口，1 或 2 个**：
  - 5 小时窗口：剩余百分比 0~100
  - 7 天（周）窗口：剩余百分比 0~100
  - 每个窗口各自带一个警戒色调：**绿 = 充裕 / 黄 = 消耗偏快 / 红 = 用尽或超限 / 灰 = 数据不可用**
  - 有的供应商只有 7 天一个窗口——**单窗口形态必须成立**，不能像缺了一块
- **数字**：剩余百分比取整、不带 % 号（现状写法 `55·96` = 周 55、5h 96；新样式可以自定数字的排布和分隔方式，语义不变）

### 组件级元素（不属于任何供应商）

- **今日用量**：今日 token 数（`775` / `3.4K` / `512.4M` 这类缩写）或今日花费（`$12.47`），二选一显示，也可整个隐藏
- 全组件无任何数据时退化为一个 15×15pt 的品牌小标（已有现成的，不用设计）

## 可定制维度（核心需求：每种样式都要能优雅退化）

用户可以在设置页做以下定制，**每种样式在任意组合下都要成立、都要好看**：

| 定制项 | 取值 |
|---|---|
| 品牌名 | 显示 / 隐藏 |
| 图形（环、条、点阵等） | 显示 / 隐藏 |
| 数字 | 显示 / 隐藏 |
| 额度窗口 | 两个都显示 / 仅 5 小时 / 仅 7 天 |
| 今日用量 | token 数 / 花费 / 隐藏 |
| 供应商 | 按家隐藏（cell 数量 1~4 家的排布都要成立） |

（品牌名 / 图形 / 数字三者至少保一个，设置页会禁止全关，不用考虑三者全无的情况。）

## 环境硬约束（菜单栏的物理现实）

- **高度**：cell 内容高度 ≤ 17pt（系统菜单栏总高约 24pt，系统留白吃掉一圈）；请按 pt 标注，@2x 出图
- **宽度寸土寸金**：单 cell 目标 ≤ 60pt、越窄越好——3~4 家 + 今日数字要同时常驻，不能挤爆菜单栏
- **底色不可控**：深色菜单栏（近黑）与浅色菜单栏（近白、可能透出壁纸颜色）两种都要能读——每种样式两种底色都要出稿
- **色彩**：只能用系统语义色——警戒三色（绿 / 黄 / 红，macOS systemGreen / systemYellow / systemRed 的气质）+ 主文字色（深底白、浅底黑）+ 次级灰。**禁止**固定品牌色（主界面的品牌青不进菜单栏）、**禁止**大面积渐变或实心色块（在菜单栏里非常违和）
- **线条**最细 1pt；**文字**最小 9pt；**数字一律等宽字体**（数值随时在变，不能横向抖动）
- 这是用户每天扫几十次的常驻组件：**一眼可读的信息层级 > 装饰性**。警戒色是信息（额度状态），不是装饰

## 每种样式需要覆盖的状态（画板组织建议）

每种样式两块画板：

1. **主画板（深色底）**：双窗口完整形态 × 四种警戒组合（全绿 / 黄绿混合 / 含红 / 全灰）+ 单窗口形态
2. **变体画板**：浅色底样例 + 至少 3 个定制退化组合（建议：「隐藏品牌名」「仅图形」「仅数字」）

最后请出一张**全景稿**：深、浅两条完整的菜单栏组件（3 家真实数据 + 今日数字并排），用来感受整体密度和节奏。

## 画稿用的真实示例数据

- Claude：7 天剩 55（黄）、5 小时剩 96（绿）
- Codex：仅 7 天窗口，剩 12（红）
- 智谱：7 天剩 78（绿）、5 小时剩 41（黄）
- 今日用量：3.4M tokens 或 $12.47

## 交付要求

- **每种样式起一个名字**（后续设置页的样式画廊会直接用）
- 矢量优先，关键尺寸用 pt 标注
- 深浅两种菜单栏底色都要有
